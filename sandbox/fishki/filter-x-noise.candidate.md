# Filter-X: «если нельзя сформулировать X — это шум» — candidate

STATUS: candidate

## CLAIM
Если мысль нельзя сжать до формулы «мы решаем X таким способом», её не стоит тащить в поток знаний.

## HOW_TO_APPLY
1) Перед ответом сформулируй X в 1 строку (проблема + контекст).
2) Если X не формулируется за 20–30 секунд — пометь как шум и не эскалируй.
3) Если X формулируется — добавь 1 проверяемый next step или артефакт.

## RISKS/LIMITS
Может отсекать сырые инсайты; смягчение — sandbox‑буфер для «сырых, но потенциально ценных» идей.

## REVIEW NOTES (edge cases)
- Исследовательские темы без чёткого X на старте.
- Междисциплинарные гипотезы, где X проявляется только после обсуждения.

## SUGGESTED EDIT
Добавить правило «2‑pass»: сначала черновой X (гипотеза), через 1 цикл — уточнённый X или архив в sandbox с причиной. Можно ввести режим “exploration” с лимитом времени/объёма до обязательной кристаллизации X.
